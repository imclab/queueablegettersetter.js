QueuableGetterSetter.js

“Like Getter Setter but Queueable”

Slides - Video - GitHub

How to use it ?

Define an Object

define an object 
var foo = { bar : 'hello' };

Define a First and a Second Getter

foo.__defineQGetter__('bar', function(value){
    return value+ ', world.';
});
foo.__defineQGetter__('bar', function(value){
    return value+ ' I love you!';
});

Use foo.bar Value

console.log(foo.bar)
// output a love message
// "Hello, world. I love you!"

How to install it ?

in node.js

$ npm install queueablegettersetter.js

in browser

<script src="queuablegettersetter.js"></script>
